# Environment  
  
The Domain that an [Agent](./Agent.md) is acting in.  
  
Many properties of environments  
- Fully Observable  
- Partially Observable  
- Single-Agent  
- Multi-Agent  
- Deterministic  
- Non-Deterministic  
- Sequential  
- Static   
- Dynamic  
- Discrete  
	- Time example  
	- A finite set of states or positions  
- Continuous
